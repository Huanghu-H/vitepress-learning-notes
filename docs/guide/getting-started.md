# 开始阅读

欢迎来到知行笔记。这是一套按主题组织的个人技术知识库，目标不是收藏所有资料，而是保留已经理解、能够复述和可以复用的内容。

## 内容地图

| 分类 | 主要内容 | 适合场景 |
| --- | --- | --- |
| Python | 语法、函数、模块、异步编程 | 复习基础与编写脚本 |
| 前端开发 | JavaScript、VitePress | 学习 Web 与维护本站 |
| 开发工具 | Git、PowerShell、终端命令 | 日常开发快速查阅 |

## 阅读方式

1. 从顶部导航进入一个学习主题。
2. 使用左侧边栏在同类文章之间切换。
3. 使用右侧目录定位当前页面的章节。
4. 按 `/` 或点击右上角搜索框搜索全文。

::: info 页面地址
VitePress 使用文件路由。例如 `docs/notes/python/basics.md` 对应 `/notes/python/basics`。
:::

## 推荐路线

如果你刚开始学习 Python，可以依次阅读：

1. [语法基础](/notes/python/basics)
2. [函数与模块](/notes/python/functions-modules)
3. [异步编程](/notes/python/asyncio)

如果你想了解本站如何搭建，请直接阅读 [VitePress 使用记录](/notes/frontend/vitepress)。

## 笔记原则

- 每篇文章只解决一个主题。
- 示例尽量保持简短且能够运行。
- 记录“为什么”，不只记录“怎么做”。
- 定期修订过时结论，避免知识库变成资料坟场。
