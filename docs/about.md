# 关于本站

知行笔记是一个用于个人知识沉淀的静态技术文档网站。内容以自己的理解为主，通过可运行示例和简短总结降低再次学习的成本。

## 技术方案

- 使用 VitePress 生成静态页面。
- 使用 Markdown 编写全部学习内容。
- 基于默认主题配置导航栏、侧边栏和本地搜索。
- 通过少量 CSS 改善中文排版和移动端表格阅读。

## 内容维护

新增笔记时，在 `docs` 下选择对应分类创建 Markdown 文件，然后在 `docs/.vitepress/config.mts` 的侧边栏中添加入口。提交前运行：

```bash
npm run build
```

构建成功表示 Markdown 可以正常编译，内部链接也通过了基础检查。
